jQuery.formatPhone
=============

jQuery plugin to format phone numbers


Usage
=============
$('selector').formatPhone({
  region:'US'
});


Options
=============
<ul>
<li><a href="#Regions">Region</a> - the locality to format the input for<br></li>
</ul>

<br><br>
<a name="Regions"/>
Currently Supported Regions:<br>
<ul>
<li>US - United States (default)</li>
</ul>
